Diffusion-Limited Aggregation

A computational physics project modelling 2D and 3D diffusion-limited aggregation (DLA) to estimate fractal dimension from simulated clusters.

What I Built

  • Implemented a 2D lattice-based DLA simulation with random walkers, adjacency sticking rules, and kill-radius logic.
  • Extended the approach to 3D with configurable seed geometry, multiple runs, and averaged scaling analysis.
  • Used log(N) vs log(r) fitting windows to estimate fractal dimension from aggregate growth.

Technical Highlights

  • Python + NumPy for simulation and counting occupied sites across radii.
  • Matplotlib visualisations for cluster structure and scaling plots.
  • Parameter controls for lattice size, particle count, spawn offset, kill factor, and run count.
